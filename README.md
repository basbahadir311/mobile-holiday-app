# Tatilin Hazır

Tatilin Hazır is a mobile holiday planning app built with React Native (Expo).

I developed this project to improve my mobile development skills and gain real experience with the Android build and deployment process.

## What I Did

- Built the mobile interface using React Native
- Managed the Android build process with Expo
- Handled configuration and build errors during development
- Published the app to Google Play Console (Closed Testing)

## Technologies

- React Native
- Expo
- JavaScript
- Google Play Console

This project helped me understand the full development cycle of a mobile application, from coding to testing and deployment.
